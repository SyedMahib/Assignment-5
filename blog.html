Question-2: What is the difference between innerHTML, innerText, and textContent?

innerText : The innerText property gives us just the text content of the element and its children. We can also add and remove the text content of an element using innerText.

innerHTML : The innerHTML property gives the HTMl content of an element. It allows us to control the inner contents such as HTMl tags by adding and removing.

textContent : The textContent propertyand the innerText property is the same. The textContent property gives us the text contents of an element without any HTML tags.
