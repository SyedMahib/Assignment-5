Question-2: What is the difference between innerHTML, innerText, and textContent?

innerText : The innerText property gives us just the text content of the element and its children. We can also add and remove the text content of an element using innerText.

innerHTML : The innerHTML property gives the HTMl content of an element. It allows us to control the inner contents such as HTMl tags by adding and removing.

textContent : The textContent propertyand the innerText property is the same. The textContent property gives us the text contents of an element without any HTML tags.

Question-3: What is event delegation in the DOM?

With event delegation we set an event listener to a parent element, by doing that it can control the events occured in its child elements. So that we dont have to set event listener individually to the child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behaviour where when we click a button and an event occures inside an element, it toches all the parent element of that element until goes up the root of the document, so the parent elements also react to that event.

Question-5: How do you create, add, and remove elements using JavaScript?

Create : To create an element we use document.createElement().
Add : To add an element we use .appendChild().
Remove : To remove an element we use the .remove()
